# Security Policy

## Supported Versions

Token Radar is pre-1.0. Security fixes are applied to the `main` branch.

## Reporting a Vulnerability

Please do not open a public issue for suspected credential exposure, local data
leaks, proxy bypasses, or provider API misuse.

Report privately through GitHub security advisories for this repository. If
that is unavailable, contact the maintainer from the GitHub profile linked to
the repository.

Useful details include:

- macOS version
- Token Radar commit or release
- provider or sync source involved
- whether local proxy capture was enabled
- minimal reproduction steps

Do not include real API keys, account tokens, provider secrets, or private
session logs in the report body.
